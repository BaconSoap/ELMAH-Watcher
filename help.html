URL Watch Pattern

Raw JS-style regex.

Code Column Pattern

Raw JS-style regex.

Notification Display Pattern

Everything inside @ symbols are column matchers - simple regex to find the column based on its text. Everything outside will be display as-is (except for @ symbols).

Filter Pattern

A list of expressions delimited by double tilde (~~) marks. A single tilde delimits column pattern (in @ symbols, like above) and the value matcher (regex, no @ symbol allowed). An error has to pass every filter to display a notification.
Exampe:
@page@~login.aspx~~@error type@~sql
